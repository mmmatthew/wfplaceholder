water-fountains.org

harnessing open data

Community project

Open data are increasingly available from both government and crowd-sourced platforms. However, the data often needs to be processed and presented visually to be of direct value to the general public. The water-fountains.org project is an open-source case study that illustrates the data-to-product process by creating a web map in which Zurich's 1200 fountains can be viewed. Data will be pulled from the official data repository data.stadt-zuerich.ch, and combined with data from wikidata and wikimedia to provide information that is both complete and rich.

The project consists in two key components: a data collector/aggregator and an interactive web map.

Component 1: Datablue

Datablue is a data collector and aggregator for open data. The fountain data, coming from multiple sources with varying degrees of reliability and containing different types of information, must be consolidated into a single consistent data set to be served to the web app. Main challenges are:

  • identifying and merging duplicate objects from different sources
  • managing multiple languages
  • concerving traceability of the information
Datablue Github project

Component 2: proximap

Proximap is an interactive map-based interface for a user to discovering fountains in their vicinity. It consumes data aggregated by Datablue and presents it in an intuitive way with powerful search functions.
