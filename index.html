water-fountains.org

harnessing open data

a community project

Open data are increasingly available from both government and crowd-sourced platforms. However, the data often needs to be processed and presented visually to be of direct value to the general public. The water-fountains.org project is an open-source case study that illustrates the data-to-product process by creating a web map in which Zurich's 1200 fountains can be viewed. Data will be pulled from the official data repository data.stadt-zuerich.ch, and combined with data from wikidata, wikimedia, and other sources to provide information that is complete, rich, and continuously maintained by its users.

Component 1: Datablue

Datablue is a data collector and aggregator for open data. The fountain data, coming from multiple sources with varying degrees of reliability and containing different types of information, must be consolidated into a single consistent data set to be served to the web app. Main challenges are:

  • identifying and merging duplicate objects from different sources
  • managing multiple languages
  • concerving traceability of the information
Datablue Github project

Component 2: proximap

Proximap is an interactive map-based interface for a user to discover fountains in their vicinity. It consumes data aggregated by Datablue and presents it in an intuitive way with powerful search and routing functions. It uses the latest web technologies to offer a seamless user experience on all devices.

background story

Back in 2015, I was on the City of Zurich's website when I stumbled across an amazing resource : a PDF file containing addresses for around 800 of the city's drinking water fountains! I was just getting interested in web mapping thanks to the inspiring work at Mapbox, so this was a perfect project to get started. I copied the data from the PDF into a text file, and found a free reverse geocoding tool online that provided approximate geographical coordinates for each water fountain address. This initial dataset was rudimentary and contained many errors, and the city of Zurich has since opened a high-quality dataset of all 1200 fountains. - Matthew Moy de Vitry
